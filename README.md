# Say Yes Learning Schools (SYLS) Website

A modern, responsive React-based website for Say Yes Learning Schools, featuring information about the institution, its board members, programs, and contact information.

## Features

- **Responsive Design**: Fully responsive website that works on all devices
- **Board Members Gallery**: Display of all 11 SYLS board members with photos
- **Multiple Pages**: Home, About, Programs, Team, Gallery, and Contact pages
- **Modern UI/UX**: Beautiful amber and gold color scheme with gradient effects
- **Contact Form**: Fully functional contact form for inquiries

## Pages

1. **Home** - Hero section with mission statement and core values
2. **About** - Information about SYLS and why choose the school
3. **Programs** - Academic, sports, arts, and life skills programs
4. **Team** - Board members gallery with professional photos
5. **Gallery** - Photo gallery categories for school events and activities
6. **Contact** - Contact form and school information

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173/`

### Building for Production

To build the website for production:
```bash
npm run build
```

The optimized files will be in the `dist/` folder.
